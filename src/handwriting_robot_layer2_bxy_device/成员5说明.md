# 成员5（bxy）：设备、视觉标定与测试材料

## 负责内容

- 调研采购写字机，安装摄像头，制作 A4 定位板和可更换笔架
- 测试中性笔、圆珠笔和不同速度，完成坐标标定
- 拍摄写后结果、计算图形误差并建立测试数据集
- 提供摄像头程序、误差报告、5 支笔 × 3 种速度参数表和演示素材

## 工作边界

设备层接收页面坐标系中的有序 strokes；设备协议和状态由成员1统一集成。不要在设备模块中复制排版或界面逻辑。

## 安全提交与进度汇报

1. 从最新 `main` 创建 `feat/bxy-device-日期-主题` 分支。
2. 代码和脱敏测试结果放在本模块；视频、原始照片和大文件使用约定的外部存储并在文档中记录位置。
3. Pull Request 写明设备型号、固件、连接方式、标定参数、测试步骤和误差。
4. 禁止提交串口密码、网络凭据、个人照片和未经允许的实验对象信息。
5. 每周汇报设备状态、标定误差、测试通过项、风险和下周计划。
