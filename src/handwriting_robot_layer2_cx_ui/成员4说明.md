# 成员4（cx）：软件界面与文档排版

## 负责内容

- 实现 Python 界面、文本输入、PDF/Word 导入和固定模板管理
- 支持手动框选书写区域、图文拖动缩放、页面预览和用户档案选择
- 提供“选择用户 → 选择模板 → 输入文字 → 导入图 → 预览 → 输出轨迹 → 开始书写”流程
- 支持任务导出、暂停和恢复操作界面

## 工作边界

界面调用系统集成层，不复制设备通信、笔迹处理或图形算法。所有输出必须交给集成层处理。

## 安全提交与进度汇报

1. 从最新 `main` 创建 `feat/cx-ui-日期-主题` 分支。
2. 只修改本模块、界面测试和必要文档；截图须脱敏。
3. Pull Request 附界面截图、操作步骤、已验证流程和已知问题。
4. 每周汇报已完成流程、可用性问题、测试环境和下周计划。
