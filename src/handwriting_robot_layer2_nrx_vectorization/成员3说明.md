# 成员3（nrx）：图形高精度矢量化

## 负责内容

- 完成 PDF/图片提取、二值化、去噪、骨架化和断线修复
- 完成直线、圆检测、曲线拟合和单线 SVG 输出
- 保留图中文字标注，支持不同图形类型参数配置
- 一期稳定支持电路图、坐标曲线、光路图或装置图中的至少三类

## 工作边界

只输出统一 `StrokeDocument`，不单独制作主界面，不直接控制设备。算法创新必须有可复现样例和误差数据。

## 安全提交与进度汇报

1. 从最新 `main` 创建 `feat/nrx-vectorization-日期-主题` 分支。
2. 代码、参数、脱敏样例和测试放在本模块；不要提交来源不明或有版权限制的原始 PDF。
3. Pull Request 中说明图形类型、输入条件、输出格式、误差和失败案例。
4. 每周汇报已支持图形、稳定性、误差指标、当前阻塞和下周计划。
