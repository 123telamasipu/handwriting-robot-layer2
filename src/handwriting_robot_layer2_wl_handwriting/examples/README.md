# Examples

- `render_request.json`：其他模块或命令行传给个人笔迹模块的渲染请求。
- `stroke_document.json`：符合 `docs/stroke-interface.md` v0.1 的最小输出示例。
- `capture_submission.json`：手机或其他采集客户端提交的统一采集数据 v1.0 示例。
- `style_generation_request.json`：标准骨架风格化请求示例。
- `demo_style_profile.json`：不包含真实用户数据的合成风格档案。
- `hanzi_writer_generation_request.json`：下载第三方汉字中心线后使用的按需生成示例。

示例输出只用于说明接口，不包含真实用户笔迹。
