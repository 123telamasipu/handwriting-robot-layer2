# 成员2（wl）：个人笔迹处理

## 负责内容

- 处理作文和笔记扫描图，分割单字并建立个人字符库
- 统计字符覆盖率，生成缺字补写清单
- 接收数位板笔迹，实现字符随机选样和大小、倾斜、间距扰动
- 尝试运行开源少样本字体模型
- 一期交付 500～1000 个样本字符、字符图片或 SVG、缺字提示和同字 2～3 种变化

## 工作边界

只输出统一 `StrokeDocument` 或字符图片/SVG，不直接控制写字机，不复制界面和设备通信代码。

## 安全提交与进度汇报

1. 从最新 `main` 创建 `feat/wl-handwriting-日期-主题` 分支。
2. 只修改本目录、对应测试和接口示例；大图、原始扫描件放在约定的外部位置，不直接提交。
3. 提交前检查是否含姓名、学号、原始隐私图片或密钥。
4. 推送后创建 Pull Request，附示例输入、示例输出、测试结果和缺字统计。
5. 每周汇报样本数量、覆盖率、已解决问题和下周计划。
